# Foad-entreprise

C’est un projet PHP pour gérer les employés d’une entreprise.
Il y a 3 boutons dans la nav :
- accueil pour revenir à la page de base
- employes pour accéder a la liste des employes avec leurs informations (id,prenom,nom etc...)
  on peut y supprimer des employes de cette liste et modifier leurs informations
- ajouter un employe comme le nom l'indique ajoute un employe a la liste complète
